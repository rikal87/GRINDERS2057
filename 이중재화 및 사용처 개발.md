# 이중 재화 및 사용처 개발 (Dual Currency & Usage)

## 1. 게임 내 시간 시스템 (Time System)
*   **비율**: 현실 시간 1초 = 게임 내 시간 1분
*   **주기**: 게임 내 24시간 = 1일
*   **시작 시점**: 2057년 10월 20일 09:00:00 (사이버펑크 세계관 몰입을 위한 설정)
*   **표시**: 인게임 UI 상단에 상시 표시. 요일 시스템 도입 시 실제 2057년 달력과 동기화.

## 2. 재화 시스템 개요 (Currency Overview)
게임은 두 가지 핵심 재화를 중심으로 경제가 순환됩니다.

### 2.1. CR (Credits)
*   **정의**: 보편적인 화폐.
*   **획득처**: 포커 게임 승리, 아이템 판매, 퀘스트 보상.
*   **사용처**:
    *   포커 테이블 바이인 (Buy-in)
    *   상점 아이템 구매
    *   **AI Agent 구독료 지불 (LT 획득을 위한 필수 지출)**
    *   안전가옥 업그레이드

### 2.2. LT (Language Token or Ludus Token)
*   **정의**: AI Agent를 구동하기 위한 연산 자원 및 해킹/네트워킹 영향력 수치.
*   **성격**: 직접적인 재화 가치보다는 "행동력(Action Point)" 또는 "마나(Mana)"에 가까운 개념.
*   **특징**:
    *   플레이어간 거래 불가능.
    *   직접적인 CR 환전 불가 (간접적 이득만 가능).
    *   안전가옥과 로비에서만 관리 및 확인 가능.
    *   **"Store(세이브 파일)"의 주요 관리 대상.**

## 3. LT 획득 및 AI Agent 구독 모델 (Acquisition & Subscription)
LT는 오직 **AI Agent**를 통해서만 생성 및 관리됩니다. 플레이어는 CR을 지불하여 AI Agent의 성능(Tier)을 구독해야 합니다.

### 3.1. 구독 등급 (Subscription Tiers)
구독료는 30일(게임 시간) 기준으로 지불됩니다. 등급이 높을수록 최대 LT 저장량과 혜택이 증가합니다.

| 등급 (Tier) | 구독료 (30일) | 최대 LT 한도 | 특징 |
| :--- | :--- | :--- | :--- |
| **LV 0 (Free)** | 0 CR | 100 LT | 기본 제공. 매우 제한적인 기능만 사용 가능. |
| **LV 1** | 10,000 CR | 300 LT | 초보자용. 기본적인 정보 수집 가능. |
| **LV 2** | 20,000 CR | 500 LT | 일반적인 해커 수준. 소규모 홈게임 개최 가능. |
| **LV 3** | 30,000 CR | 1,000 LT | 숙련자용. 중급 스킬 사용 가능. |
| **LV 4** | 50,000 CR | 2,500 LT | 전문가용. 고급 해킹 및 VIP 구역 접근 가능. |
| **LV 5** | 100,000 CR | 4,000 LT | 엘리트용. 카지노 시스템에 직접 간섭 가능. |
| **LV 6** | 200,000 CR | 5,500 LT | 레전드용. 도시 전역에 영향력을 행사. |

### 3.2. 재생성 (Regeneration)
*   **자연 회복**: 게임 시간 1시간당 **최대 LT 한도의 1%** 충전.
    *   *예: LV 3(1,000 LT)는 시간당 10 LT 회복.*
*   **부스트 충전**: (기획 필요) 특정 소모품(냉각수, 배터리 등)을 사용하여 일시적으로 급속 충전 가능?

## 4. LT 사용처 (Usage & Effects)
LT는 "AI Agent 스킬(Task)"을 실행하는 데 소모됩니다. 이 스킬들은 게임의 **메타(Meta)** 요소에 영향을 줍니다.

### 4.1. 환경 조작 (Environment Manipulation)
*   **테이블 커스터마이징**: 특정 지역(Zone)의 플레이어 성향(Persona) 분포 변경. (예: Fish 비율 증가)
*   **레이크(Rake) 해킹**: 특정 지역의 레이크 요율 일시적 감소. (최대 0%까지)
*   **입장 권한 해킹**: 높은 명성치가 필요한 VIP 라운지(Orbit Lounge 등) 일시적 출입 권한 획득.

### 4.2. 수익 효율화 (Profit Efficiency)
*   **암시장 협상**: 아이템 재판매 시 CR 환급률 증가 (기본 25% -> 최대 50%).
*   **경험치 부스트**: 일정 시간 동안 획득 XP 증가.

### 4.3. 정보 수집 (Intel Gathering)
*   **상대 프로파일링**: 테이블 내 특정 플레이어의 상세 성향(vPIP, PFR 등) 미리보기.
*   **핸드 히스토리 저장**: 플레이한 핸드 데이터를 안전가옥 서버에 영구 저장하여 복기 가능하게 함.

### 4.4. 네트워킹 (Networking)
*   **홈게임 개최**: 지인(특정 페르소나군)을 초대한 비공개 테이블 생성. (레이크 0%, 특정 블라인드 설정 가능)

## 5. 설계 원칙 (Design Principles)
1.  **NO Pay-to-Win (Directly)**: LT를 써서 포커 핸드 승률을 직접 조작(예: 리버에 A가 뜨게 함)하는 것은 절대 금지.
2.  **Meta-Game Influence**: LT는 게임을 둘러싼 '환경'을 유리하게 만드는 데 사용됨.
3.  **Strategic Resource**: LT는 한정된 자원이므로, 플레이어는 언제 어떤 스킬을 켜서 효율을 극대화할지 전략적으로 선택해야 함.
