
### 1. 모바일 버전 UI 개선
작은 해상도에서 테이블 UI가 비좁은 문제를 개선하기위해 원근감 시스템을 도입하여 비스듬하게 배치되도록 변경 멀리 있는 상대가 작아보이게됨

### 2. AI에이전트 시스템 리뉴얼
이제 기존의 7일마다 돈을 소진하는 구독 형태로 제공되던 에이전트가 영구적으로 업그레이드 하는 형태로 변경되었습니다.
또한 레벨 도달시 잠금해제 되는 에이전트의 첫 티어는 항상 "무료"로 제공됩니다.

### 3. 경제 관련
에이전트를 활용하여 돈을 버는 AGENT_WORK 계열의 작업들의 수익이 전체적으로 크게 증가하였습니다.
파산시 뱅크롤을 지급하는 아이템의 성능이 변경되었습니다. 기존에는 완전히 파산했을때 일정금액을 지급하는 방식이였으나, 이제 테이블에서 초기 바이인을 모두 잃을경우 초기 바이인에 일정비율을 환급해주는 방식으로 변경되었습니다.
