# LT 기반 스킬 (Task) 상세 설계안

## 1. 개요 (Overview)
AI Agent는 LT를 소모하여 특정 작업(Task)을 수행합니다. 이를 "스킬"이라고 부르며, 플레이어는 자신의 플레이 스타일(그라인딩, 한탕주의, 정보수집 등)에 맞춰 스킬을 조합할 수 있습니다.

## 2. 기본 메커니즘 (Mechanics)
*   **슬롯 (Memory Slots)**: AI Agent 등급에 따라 동시에 활성화할 수 있는 스킬의 개수가 제한됩니다.
    *   LV 0: 1개
    *   LV 1~2: 2개
    *   LV 3~4: 3개
    *   LV 5~6: 4개
*   **비용 (Cost)**: 작업 수행 시 **시간당 LT**가 지속적으로 소모됩니다.
*   **확률 (Probability)**: 작업 수행 중 일정 확률로 효과가 발동합니다. 실패 시에도 LT는 소모되지만, 실패할수록 성공 확률 보정치(Bad Luck Protection)가 적용됩니다.
*   **발동 효과 (Effects)**: 발동에 성공하면 즉시 효과가 적용되며, 효과 지속 시간 동안은 해당 작업 수행이 중단(비활성화)됩니다.
*   **쿨타임 (Cooldown)**: 효과 종료 후 해당 스킬을 다시 활성화하기까지 필요한 대기 시간입니다.

## 3. 스킬 목록 (Skill List)

### [TIER 1] - 입문자용 (Starter)

#### **1-1. 동네 포커 모임 (Neighborhood Game)**
*   **유형**: 네트워킹
*   **설명**: 지인들을 불러 모아 레이크가 없는 홈게임을 엽니다.
*   **비용**: 1 LT / 시간
*   **발동 확률**: 10% / 시간 (매 시간 10% 확률로 개최 성공)
*   **발동 효과**: 
    *   **레이크 0%** 테이블 생성 이벤트 발생 (메시지함으로 초대장 발송).
    *   참가자: 무작위 [Fish, Rock, Calling_Station] 혼합.
    *   제한: 블라인드 5/10 CR 고정.
*   **지속시간**: 최대 4시간 (테이블 유지 시간).
*   **쿨타임**: 3일

---

### [TIER 2] - 일반 (Common)

#### **2-1. 쉬운 먹잇감 탐색 (Fish Finder)**
*   **유형**: 탐색
*   **설명**: "쉽게 돈을 벌려면 소프트해 보이는 테이블을 찾아야 합니다!"
*   **비용**: 3 LT / 시간
*   **발동 확률**: 10% / 시간
*   **발동 효과**: 
    *   현재 구역(Zone)의 신규 테이블 생성 시 **Fish** 페르소나 등장 확률 2배 증가.
    *   (조건: 해당 구역 등장 풀에 Fish가 존재해야 함)
*   **지속시간**: 1일
*   **쿨타임**: 3일

#### **2-2. 광란의 테이블 (Maniac Attractor)**
*   **유형**: 탐색
*   **설명**: "미친 놈들과 춤을 추고 싶다면, 이 주파수를 켜세요."
*   **비용**: 4 LT / 시간
*   **발동 확률**: 10% / 시간
*   **발동 효과**: 
    *   **Maniac** 페르소나 등장 확률 3배 증가. 
    *   게임의 변동성(Variance)이 극도로 높아집니다.
    *   (조건: 해당 구역 등장 풀에 Maniac이 존재해야 함)
*   **지속시간**: 1일
*   **쿨타임**: 3일

#### **2-3. 포커 유망주 육성 (Grinder's Mindset)**
*   **유형**: 부스트
*   **설명**: 극한의 집중력을 발휘하여 학습 능력을 극대화합니다.
*   **비용**: 50 LT (일시불)
*   **발동 확률**: 100% (즉시 발동)
*   **발동 효과**: 
    *   획득 XP +25%.
*   **지속시간**: 1일
*   **쿨타임**: 없음 (지속시간 종료 후 바로 재사용 가능)

---

### [TIER 3] - 숙련자 (Uncommon)

#### **3-1. 부자 털어먹기 (Rich Guy Hunter)**
*   **유형**: 탐색
*   **설명**: "30분 내에 호구를 못 찾으면 내가 호구라지만, 난 이미 찾았는걸?"
*   **비용**: 8 LT / 시간
*   **발동 확률**: 10% / 시간
*   **발동 효과**: 
    *   **Rich_Guy** (돈 많은 호구) 등장 확률 3배 증가.
    *   (조건: 해당 구역 등장 풀에 Rich_Guy가 존재해야 함)
*   **리스크**: **Shark** (고수)가 10% 확률로 테이블에 추가됨 (냄새를 맡고 옴).
*   **지속시간**: 1일
*   **쿨타임**: 10일

#### **3-2. 핸드 히스토리 마이닝 (Hand History Mining)**
*   **유형**: 정보
*   **설명**: 테이블의 모든 핸드 내역을 암호화하여 저장하고 분석합니다.
*   **비용**: 50 LT (일시불)
*   **발동 확률**: 100% (즉시 발동)
*   **발동 효과**: 
    *   플레이한 모든 핸드 히스토리 **영구 저장** (복기 가능).
    *   상대방의 주요 스탯(vPIP, PFR 등)이 HUD에 상세 표시됨.
*   **지속시간**: 1일
*   **쿨타임**: 없음

---

### [TIER 4] - 전문가 (Rare)

#### **4-1. 오르빗 네트워크 해킹 (Orbit Network Hack)**
*   **유형**: 해킹/잠입
*   **설명**: "출입 금지? 그건 일반인들 얘기고."
*   **비용**: 100 LT / 시간 (작업 수행 중 비용)
*   **발동 확률**: 5% / 시간
*   **발동 효과**: 
    *   명성치 제한 없이 **Orbit Lounge** 등 High Stakes 구역 입장 권한 획득.
*   **리스크**: 작업 수행 중 매 시간 1% 확률로 보안팀에 발각.
    *   발각 시: 작업 취소, 강제 접속 종료, 벌금 **100,000 CR**.
*   **지속시간**: 1일
*   **쿨타임**: 20일

---

### [TIER 5] - 전설 (Legendary)

#### **5-1. 실력자 초청 게임 (Pro's Table)**
*   **유형**: 네트워킹
*   **설명**: "진정한 승부를 원해? 전설들을 불러주지."
*   **비용**: 100 LT / 시간
*   **발동 확률**: 5% / 시간
*   **발동 효과**: 
    *   [Old_Lion, Shark, GTO_Bot] 등 최상위 페르소나들로 구성된 하이롤러 홈게임 개최.
    *   승리 시 막대한 명성치 및 희귀 아이템 획득 기회.
*   **지속시간**: 1일 (테이블 유지)
*   **쿨타임**: 20일

#### **5-2. 카지노 메인프레임 장악 (The 'House' Always Wins?)**
*   **유형**: 해킹
*   **설명**: 일시적으로 당신이 '하우스'가 되어 시스템을 조작합니다.
*   **비용**: 100 LT / 시간
*   **발동 확률**: 5% / 시간
*   **발동 효과**: 
    *   해당 세션 동안 플레이어가 지불하는 **레이크 0%**.
*   **지속시간**: 1일
*   **쿨타임**: 3일

## 4. UI/UX 구현 고려사항
*   **Skill Dashboard**: 안전가옥 PC 또는 휴대 단말기(UI)에서 스킬 장착/해제.
*   **Status Indicator**: 현재 활성화된 스킬, 남은 지속시간, 현재 LT 소모량 표시.
*   **Notifications**: 스킬 발동 성공 시, 리스크 발생 시(해킹 발각 등) 시스템 메시지 알림.
