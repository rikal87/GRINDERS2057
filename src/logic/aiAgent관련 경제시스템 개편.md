# AI Agent 경제 시스템 개편안 (최종)

## 1. 시스템 개요
기존의 기간제 구독 시스템을 폐지하고, 각 AI 모델을 영구적으로 소유하고 강화하는 **영구 업그레이드 시스템**으로 전환합니다.

## 2. 주요 변경 사항

### 2.1. 구독 시스템 폐기
*   7일 주기 자동 갱신 및 구독료 지불 시스템 삭제
*   구독 만료(`subscriptionExpireAt`) 개념 삭제
*   모든 AI 모델은 한 번 획득/업그레이드하면 영구적으로 효과가 유지됨

### 2.2. 순차적 업그레이드 시스템
*   기존의 등급(Tier)은 이제 영구 업그레이드 단계로 전환됨
*   **순차적 증가 강제**: 하위 단계를 건너뛰고 상위 단계를 바로 구매할 수 없음 (예: Tier 0 -> Tier 1 -> Tier 2 순서로만 가능)
*   각 AI 모델의 첫 번째 플랜(Tier 0)은 항상 무료로 제공됨

### 2.3. 모델별 데이터 보존 및 전환
*   **개별 진행도 저장**: 여러 개의 AI 모델을 보유할 수 있으며, 각 모델의 업그레이드 수치는 개별적으로 보존됨
*   **자유로운 교체**: 사용 중인 AI 모델을 다른 모델로 변경하더라도, 기존 모델의 업그레이드 정보는 소멸되지 않고 그대로 유지됨
*   교체 시 해당 모델의 현재 티어에 맞는 슬롯 및 능력치가 즉시 적용됨

## 3. 기술적 구현 가이드
*   `store.js` 내에 모델별 티어를 저장하는 `aiAgentTiers` 객체 도입
*   `aiAgentTaskSystem.js`에서 구독 체크 로직 제거 및 업그레이드/교체 기능 구현
