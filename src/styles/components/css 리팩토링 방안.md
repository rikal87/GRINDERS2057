1. 현재 mobile-*.css은 모바일디바이스 전용 css이며 데스크탑에서 중복 선언이있기 때문에 해당 부분은 분리하여 desktop-*.css에 이동한다.
2. 데스크탑이나 모바일 해상도에 따른 레이아웃 변경 관련 css는 별도 파일로 분리하여 작업한다.
3. 디바이스 상관없이 공통으로 사용되는 스타일은 별도 파일로 분리한다. (비교분석을 위해 현재 파일 보존 권장)
4. 컴포넌트 별로 css를 개별 임포트해서 사요한다.
5. 글로벌 css는 전역에서 사용되는 컴포넌트이니 메인에 임포트한다.
6. 현재 데스크탑 버전으로 최적화된 css는 가급적 원본을 유지하라
