# 안전 가옥 메시지함 (Safehouse Message Inbox) 상세 설계

## 1. 개요 (Overview)
안전 가옥의 메시지함은 플레이어가 게임 세계와 상호작용하는 비동기 커뮤니케이션 채널입니다. 단순한 알림을 넘어, 게임의 경제적 압박(세금, 임대료)과 서사적 몰입감(스팸, 뉴스)을 전달하는 핵심 UI입니다.

## 2. 데이터 구조 (Data Structure)
메시지는 다음과 같은 구조를 가집니다.

```json
{
  "id": "msg_unique_id",
  "type": "SYSTEM | FINANCE | SOCIAL | SPAM | TIER_EVENT | TUTORIAL",
  "sender": "Sender Name (e.g., 'NC City Admin', 'Unknown')",
  "title": "Message Title",
  "body": "Message Content (HTML support for coloring/bold)",
  "timestamp": 2773123456, // In-game timestamp
  "isRead": false,
  "actions": [ // Optional interactive buttons
    {
      "label": "납부하기 (10,000 CR)",
      "actionType": "PAY_RENT | PAY_TAX | ACCEPT_INVITE | TUTORIAL_MISSION",
      "payload": { "amount": 10000, "target": "rent" }
    }
  ],
  "expireAt": 2775123456 // Optional expiration time for time-sensitive tasks
}
```

## 3. 메시지 유형 및 트리거 (Message Types & Triggers)

### 3.1. 재정 (Finance) - **CRITICAL**
플레이어의 생존과 직결된 메시지입니다. 무시할 경우 게임 오버로 이어질 수 있습니다.

#### **A. 안전가옥 임대료 (Rent)**
*   **발송 주기**: 매월 1일 (게임 시간 30일 주기).
*   **내용**: "10월분 안전가옥 임대료 청구서입니다."
*   **기능**: [납부하기] 버튼 포함.
*   **페널티**:
    *   30일 연체: 경고 메시지.
    *   60일 연체: "퇴거 예고" 및 일부 기능 제한.
    *   90일 연체: **GAME OVER** (강제 퇴거 및 파산 처리).

#### **B. 소득세 (Income Tax)**
*   **발송 주기**: 매주 월요일 (게임 시간 7일 주기).
*   **계산식**: `(지난 주 총 포커 수익 - 지난 주 손실) * 세율(등급별 차등)`
    *   *손실은 차감 계산되나, 마이너스일 경우 세금은 0원.*
*   **내용**: "귀하의 지난 주 소득에 대한 납세 고지서입니다."
*   **페널티**: 미납 시 신용도 하락, 카지노 출입 제한, 심할 경우 강제 징수(계좌 압류).

### 3.2. 시스템 및 이벤트 (System & Event)
플레이어의 행동 결과와 게임의 진행 상황을 알리는 동적 메시지입니다.

#### **C. 작업(Task) 결과 알림**
*   **개요**: AI Agent가 수행한 확률형 작업의 결과를 통보합니다.

    1.  **홈 게임 개최 준비 완료 (성공 시)**
        *   **트리거**: '동네 포커 모임', '실력자 초청 게임' 스킬 발동 성공 시.
        *   **발신자**: AI Agent (My Assistant)
        *   **제목**: [Task] 홈 게임 호스팅 준비 완료
        *   **내용**: "요청하신 홈 게임 참가자 모집이 완료되었습니다. 지금 바로 게임을 시작하시겠습니까? (유효시간: 4시간)"
        *   **액션 버튼**:
            *   **[지금 시작하기]**: 즉시 안전가옥 테이블 화면으로 이동하여 게임 시작.
        *   **만료 시**: "참가자들이 기다리다 지쳐 떠났습니다." 메시지로 변경되며 실행 불가.

    2.  **해킹 성공 (성공 시)**
        *   **트리거**: '권한 해킹', '정보 수집' 스킬 발동 성공 시.
        *   **발신자**: Unknown (Anonymous)
        *   **제목**: [Intel] 오르빗 라운지 임시 접근 권한
        *   **내용**: "백도어 코드가 심어졌습니다. 24시간 동안 보안 시스템이 귀하를 VIP로 인식합니다."
        *   **효과**: 오르빗 라운지 입장가능
        *   **액션 버튼**:
            *   **[승락]**:  메시지 수신 즉시 해당 효과(버프) 적용 시작.
        *   **만료 시**: "오르빗 라운지 임시 접근 권한이 만료되었습니다." 메시지로 변경되며 승락 불가

    3.  **해킹 실패 및 경고 (실패/리스크 발동 시)**
        *   **트리거**: 해킹 스킬 사용 중 '보안 팀 발각' 리스크 발생 시.
        *   **발신자**: S7PD (SECTOR 7 Police Dept.)
        *   **제목**: [WARNING] 불법 접속 시도 감지
        *   **내용**: "귀하의 단말기에서 카지노 서버로의 비정상적인 접속 시도가 포착되었습니다. 과태료가 부과됩니다."
        *   **액션 버튼**: **[벌금 납부 (100,000 CR)]** (미납 시 수배 레벨 상승).


#### **E. 튜토리얼 및 가이드**(추후 개발 예정)
*   **개요**: 게임 초기 정착을 돕는 단계별 가이드 메시지.
*   **상세 내용**: 별도 문서 [튜토리얼 상세 기획안](튜토리얼.md) 참고.

### 3.3. 로어 및 스팸 (Lore & Spam)
사이버펑크 세계관의 분위기를 살리는 플레이버 텍스트입니다. 기능적 효과는 없습니다.
** 관련 실제 데이터는 lore_spam_message.js를 참고 **

#### **F. 시티 뉴스 / 트리비아 / 스팸**
*   **개요**: 나이트 시티의 뉴스, 광고, 스팸 메시지 등.

## 4. UI/UX 디자인 가이드
*   **위치**: 안전가옥 컴퓨터 터미널 내 [Inbox] 탭.
*   **알림 표시(Badge)**: 로비나 안전가옥 화면 구석에 읽지 않은 '중요' 메시지(재정, 퀘스트) 수 표시. (스팸은 카운트 제외 옵션)
*   **리스트 뷰**: 보낸 사람, 제목, 날짜, 읽음 여부 아이콘.
    *   중요/긴급 메시지는 **붉은색 텍스트**나 **[!] 아이콘**으로 리스트 상단 고정.
*   **상세 뷰**: 
    *   좌측: 메시지 리스트 / 우측: 선택된 메시지 내용 (분할 패널 추천).
    *   하단: 액션 버튼 영역 (버튼이 필요한 경우에만 표시).

## 5. 향후 확장 (To-Be)
*   **암호화폐 거래소 정보**: 특정 코인의 급등/급락 찌라시 정보를 메시지로 제공 (유료 정보상).
*   **NPC 상호작용**: 특정 NPC가 보내는 갠톡/미션 의뢰.
