해당 UI에선 아래와 같은 UI가 제공됨(FLEX-FLOW: ROW)


1. 모델을 대표하는 이미지 [DUMMY 삽입]
2. AI대한 설명이 제공됨 [slogan, key_features]

4. 결제 버튼


---------결제 프로세스 -----------

1. 팝업 UI에서 선택한 모델과 구독 플랜을 store에 저장
2. 구독료는 선택한 즉시 CR에서 차감
3. 다음 구독료는 게임시간으로 매 30일마다 자동 인출되며, 만약 돈이 부족할경우 구독이 해지되며 VANGUARD AGENT의 FREE TIER (0)로 강제 전환 (인박스 메시지로 해지 알림 메시지 전송)


--------- 요금제 강제 해지시 ------------
1. TASK_SLOT에 할당되었던 TASK는 일단 유지되나 기본 에이전트의 슬롯 갯수가 작업중인 TASK 슬롯보다 부족한 경우 가장 아래부터 비활성화
2. 비활성화된 TASK_SLOT에 있는 TASK는 작업이 비활성화됨
